# cli-logs
A useful tools for logging thing in the terminal

# Examples
```js
const cl = require('cli-logs');

cl.ready('Node is ready!');
```

# Available Options
```js
const cl = require('cli-logs');

cl.ready();
cl.error();
cl.warn();
cl.event();
cl.load();
cl.connect();
cl.debug();
cl.cmd();
```
More options will be added soon!
